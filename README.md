 🎮 Unity Essential Project

This project was created as part of the Unity Essentials Pathway. It demonstrates my understanding of the Unity Editor, 3D scene design, and basic game mechanics.


 🚀 Features

* 3D environment navigation
* Object movement and rotation
* Interactive elements (robot collecting objects)
* Scene exploration and camera control


🧠 What I Learned

* Unity Editor interface and tools
* Scene view navigation and controls
* GameObjects and components
* Transform system (position, rotation, scale)
* Basics of player interaction


🛠️ Technologies Used

* Unity (Game Engine)
* C#
* Git & GitHub




🔮 Future Improvements

* Improve lighting and environment design
* Add more gameplay mechanics
